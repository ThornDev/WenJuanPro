## Risk Coverage

| Risk | Mitigated By |
|------|-------------|
| Data loss on crash | 2.5-UNIT-010, 2.5-INT-007 (flush+fsync verification) |
| Corrupt resume file | 2.5-UNIT-017, 2.5-INT-015 (skip corrupt, log warning) |
| Half-line write | 2.5-UNIT-009 (atomic full-line write + newline) |

| File handle leak | 2.5-INT-006, 2.5-BLIND-RESOURCE-001 |
| Resume picks wrong file | 2.5-INT-013 (latest timestamp selection) |

5. P1 Blind Spots (BLIND-BOUNDARY-001, 003; BLIND-ERROR-001~002; BLIND-RESOURCE-001)
6. P2 Blind Spots (remaining)
