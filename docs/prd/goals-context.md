# 目标与背景

## 目标

- 在 2026 Q3 前交付 MVP，支撑至少 2 个真实科研课题完成首轮线下离线认知测评。
- 通过"扫码即作答 + TXT 即配置 + 即时落盘"三件套，将研究员从拿到 APK 到完成首轮回收的端到端首次成功周期压缩到 ≤ 1 个工作日。
- 单台设备稳定承载 ≥ 200 名学生连续作答，数据完整率 ≥ 99.5%、记忆题视觉时序抖动 ≤ ±50ms、断点续答命中率 ≥ 95%。
- 提供差异化的「记忆闪烁点击题」范式，验证"阶段化呈现 + 空间复现"在 Compose 原生动画 API 上的工程可行性，为后续认知范式扩展奠基。
- 流程零账号、零云、零网络依赖，确保未成年人测评数据**不出设备**。

## 背景

国内高校与基础教育研究现场普遍处于弱/无网络环境，主流问卷工具强依赖在线服务，且不支持基于倒计时与顺序复现的视觉认知任务；现有心理学实验软件以 PC 桌面为主，难以在现场规模化发放手机/平板。研究员侧的真实诉求是：**不学新工具、直接改 TXT、扫码即作答、TXT 即结果**。

WenJuanPro 以 Android 原生 + Jetpack Compose 原生动画 API 为载体，将题库、素材与结果全部落在 `/sdcard/WenJuanPro/` 三个固定子目录下；APK 仅手动侧载分发，因此可放心使用 `MANAGE_EXTERNAL_STORAGE` 直接读写；学生扫码（二维码即学号）即进入作答，会话异常中断后下次扫码同一学号自动续答。MVP 仅交付三种题型（单选 / 多选 / 记忆闪烁点击题）与单一阶段化呈现模式，刻意抑制 GUI 配题工具、云同步、统计可视化等延伸需求，将工程预算集中到"离线可靠 + 时序稳定 + 数据可追溯"三件最难做对的事上。
