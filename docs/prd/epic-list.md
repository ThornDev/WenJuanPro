# Epic 列表

- **Epic 1: 项目基础与配置加载** — 建立 Android 工程骨架、`MANAGE_EXTERNAL_STORAGE` 授权流、TXT config 解析与校验、测评集选择页（首屏即可见可用测评集，研究员可立即验证 TXT 配置正确性）。
- **Epic 2: 扫码登录与基础题型作答** — 实现扫码 → 欢迎确认 → 单选/多选题作答（含同屏与分阶段模式倒计时）→ 结果实时落盘的端到端主流程（无记忆题、无续答时即可完成一份"普通问卷"测评）。
- **Epic 3: 记忆闪烁点击题** — 实现 8×8 点阵 + 闪烁动画序列 + 1.0× 半径命中判定 + 顺序复现记录的完整记忆题作答闭环，达成 ±50ms 时序抖动目标。
- **Epic 4: 断点续答** — 实现同学号会话发现、已答题跳过、半完成题整题重做策略，达成 ≥ 95% 续答命中率与 100% 续答位置恢复。

> **理由**: 4 个 Epic 按"基础 → 普通问卷 → 差异化范式 → 可靠性"递进，每个 Epic 完成后均可独立部署并交付可见价值（Epic 1 = 研究员可校验配置；Epic 2 = 可跑通完整非记忆题问卷；Epic 3 = 可完成全部题型；Epic 4 = 现场可承诺零数据丢失）。Epic 1 同时承担基础设施 + 首个可见功能（测评集选择页），符合 agile 最佳实践。
