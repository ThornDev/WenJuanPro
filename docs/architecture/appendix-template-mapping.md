为便于追溯本文档与 `fullstack-architecture-tmpl` 的对应关系，下表列出主要偏离点：

| 模板章节                      | 本文档处理                                                                                      |
|-------------------------------|-------------------------------------------------------------------------------------------------|
| API 规格 (OpenAPI/GraphQL/tRPC) | 折叠为 N/A；等价物改为「文件存储 Schema」节                                                      |
