**As a** 学生（被试），
**I want** 续答时系统自动跳过已完成的题目，直接从上次中断的地方继续，
**so that** 我不需要重复作答已完成的题目。

## Implementation Note

核心功能已在现有代码中支持：
- `QuestionViewModel` 通过 `cursor` 控制当前题目索引，续答时 `cursor = completedQids.size`
- `StartSessionUseCase` 支持 `cursor` 参数恢复
- `ResultFileParser.parseCompletedQids()` 已更新为严格按 status=done 过滤（Story 4.1 修改）
- `ResultFileParser.parseQidsByStatus()` 新增，返回 qid→status 映射，用于漂移检测

本故事的新增代码仅为 config 漂移检测逻辑。
